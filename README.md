# FoodGenie - Restaurant Management System

FoodGenie is a MERN stack (MongoDB, Express.js, React.js, Node.js) restaurant management system built during a Full Stack Development internship.

## Key Features
- Browse restaurants and discover menus
- Search, filter, and sort restaurants and food items
- Secure user registration and login with JWT authentication
- Cart management and end-to-end food ordering flow
- Menu management and order processing

## Tech Stack
- **Frontend:** React.js
- **Backend:** Node.js, Express.js (REST APIs)
- **Database:** MongoDB
- **State Management:** Redux Toolkit
- **Authentication:** JWT

## Work Done
- Designed and implemented frontend and backend modules
- Built responsive React components and integrated backend services
- Developed RESTful APIs with Express.js
- Modeled and integrated MongoDB data storage
- Implemented authentication, state management, cart, and ordering workflows
- Debugged issues, tested APIs, and maintained code with Git/GitHub

## Outcomes
This project improved practical skills in full-stack architecture, API design, database modeling, authentication, state management, debugging, and collaborative development.
